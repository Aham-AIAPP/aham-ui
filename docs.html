02设计性格北极星:Claude desktop 气质
01
冷色的纸

三层中性灰,无暖调。界面像一张安静的纸,内容是主角,背景退后。

02
克制的金属感

扁平、精确、无材质。深度靠层差不靠模糊;蓝是点缀不是填充。

03
对话式

对标 Claude desktop。安静、留白优先、内容先行,工具退到注意力之外。

清晰 Clarity — 内容是主角,文字易读,层级分明。

克制 Restraint — 默认是「不加」。每个颜色/字体/装饰都要有理由。

层次 Depth — 深度靠三层背景层差,绝不靠材质/模糊/阴影。

铁规速览
  • 状态 = 6px 点 + 文字,绝不 pill / 色块 / 红黄绿灯
  • 表格只横线,无竖线无整行底色;数字右对齐 mono;表头/合计加粗
  • 卡片无边框无阴影;选中 = 扁平灰非蓝;静置/hover 无阴影,仅浮层有阴影
  • 蓝 #336EE8 只用于 logo / 主操作 / 发送 / 选中——第一眼注意到就超标
  • 单一无衬线(Inter),数字 mono;不靠颜色单独传达(多类别用形状/文字/图标)
